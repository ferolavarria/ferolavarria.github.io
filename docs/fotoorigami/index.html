Fotografía Molde Origami

Kuchen de Yogurt

Este kuchen de yogurt de forma distinta a la de constumbre está divido y relizado en dos moldes diferentes. Primero, la base de bizcocho y la capa de yogurt, están realizados en un molde rectangular (sin origami), donde el bizcocho fue llevado al horno y la capa de yogurt al refrigerador. Posteriormente se relalizó el molde con el patrón de Origami, el cual está fabricado de cartón forrado, donde posteriormente se forró utilizando papel aluminio para que no se filtrase el líquido. Los resultados no fueron los esperados al 100%, ya que el molde no resistía tanto peso y se desformó (volviendo a la forma plana) durante el proceso, aún así, se pueden observar algunas formas del patrón mismo.

Brownie

Este Brownie fue realizado en un molde fabricado por Rhinoceros, mediante Grasshopper. Se forma a partir del plegado de un pentágono. El molde fue forrado en papel mantequilla como desmoldante y fue sometido a 180°C durante 20-25 minutos. El molde fue fabricado con cartón forrado, al igual que el molde anterior. Buenos resultados con este.
