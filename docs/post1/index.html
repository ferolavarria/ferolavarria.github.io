Proyecto Final Taller de Diseño de Interfaces

Descripción

Proyecto creado como evaluación final del Taller de Diseño 2020. Sauka (o 5-VID) es un sanitizador de efectivo para los pequeños negocios de barrio, que mediante luz UV logra eliminar la mayor parte de virus y bacterias que poseen los billetes y monedas. Se brinda una mayor seguridad para las personas al momento de realizar intercambiar dinero cuando se efectúa una compra en un pequeño negocio, donde la mayor parte de pagos se realiza con dinero físico (sin contar con método de pago de tarjetas de crédito, tarjetas de débito o transferencias). Este proyecto fue pensado para ayudar a disminuir la transmisión del virus COVID-19 al momento de efectuar una compra en los locales menores (de barrio).
